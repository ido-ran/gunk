Interfacepublic interface IKey extends IHashObject
Implementors Key

A key is used in two places:
  1. To uniquely identify a binding
  2. To uniquely identify a dependency

The hash of a key represents the type and the names of the annotations. The full hash also takes the default value and values of the annotation into account.
